Admin Instructions

1. How to Add a New App

  1. Click on Add / Edit App in the sidebar.
  2. Fill in the fields. App ID must be unique (e.g., 'app-name-mod').
  3. Icon URL: Right-click an image on Google -> 'Copy Image Address' and paste it here.
  4. Click Add App.

2. SEO Settings

These fields help your app appear in Google search results.
SEO Title: This appears in the browser tab and search link.
SEO Desc: A short summary shown below the link in Google actions. SEO Desc: A short summary shown below the link in Google search results.

3. CRITICAL: How to Save Changes

The changes you make here are temporary until you save them to the file.
1. Standard Method: Click Save Changes (data.js). Replace the `data.js` file in your project folder.
2. JSON Method: If you want a raw JSON file, click Export as JSON.
3. Refresh your website (`index.html`) to see the updates!
